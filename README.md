# AutSecurity Box Writeups

Bienvenido al repositorio oficial de writeups de **AutSecurity Box**, una plataforma de entrenamiento en ciberseguridad diseñada para desarrollar habilidades ofensivas y defensivas mediante laboratorios prácticos y escenarios realistas.

## ¿Qué encontrarás aquí?

Este repositorio contiene documentación técnica detallada de las máquinas y desafíos publicados en AutSecurity Box, incluyendo:

* Enumeración de servicios y reconocimiento.
* Identificación y explotación de vulnerabilidades.
* Escalada de privilegios en sistemas Linux y Windows.
* Técnicas de movimiento lateral y post-explotación.
* Active Directory y entornos empresariales.
* Aplicaciones web vulnerables.
* Buenas prácticas y metodologías utilizadas durante el proceso de compromiso.

## Objetivo

El propósito de estos writeups es reforzar el aprendizaje práctico, ayudar a la comunidad a comprender diferentes vectores de ataque y servir como material de estudio para estudiantes, profesionales de seguridad, pentesters y entusiastas de la ciberseguridad.

## Importante

Todo el contenido publicado en este repositorio está destinado exclusivamente a fines educativos y de investigación dentro de entornos controlados. Las técnicas descritas deben utilizarse únicamente en sistemas autorizados.

## Acerca de AutSecurity Box

AutSecurity Box es una plataforma enfocada en la formación práctica de profesionales de ciberseguridad mediante laboratorios diseñados para simular escenarios reales/ctfs, permitiendo a los usuarios desarrollar competencias técnicas en reconocimiento, explotación, escalada de privilegios y análisis de seguridad.
